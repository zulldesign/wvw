Installation

  • Upload all content of package in IIS, recommended don't rename the folder blogdotnet
  • Go to blogdotnet/setup/Default.aspx
  • Create a database in MySql Server
  • Execute queryblog.sql, in setup folder with phpmyadmin or another MySql manager sofware
  • Modify config.inc to reflect your data in MySql Server
  • Go to second step /yourblogfolder/Default1.aspx and enter your Data
    • Admin Nick: your nick as Administrator
    • Password: your password for admin section and blog section
    • Email: your email administrator
    • Blog Title: the title of your blog
    • Location
    • Email Server: your mail server used to email notification - example smtp.provider.com
  • Click on Setup the blog, after this the process is complete
  • Grant write permissions on admin/upload folder
  • Remove the setup folder
