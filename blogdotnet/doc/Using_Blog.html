How works the Blog

  • Setting Information of blog
On the file latsx.html writes your information of blog; author, argument and so on(necessary minimal html knowledge)


  • Managing the news and the users
Enter in /blogfolder/admin/default.aspx and login with the userid and password setted during installation process.
Once logged you can access on /blogfolder/admin/index.aspx directly; after this you can log in the blog and post and remove comments(you are Superuser) of blog.

- To manage administrator data go to Manage admin Data

- To manage Categories click on Manage Categories, you can change the name of category or remove It

- To insert the news go to Manage News --> Insert News
insert the news together the category, choice if you want receive email notification when an user post a new message
To delete or change the news go to Manage News --> Delete or Change News

To manage Users clicks on Manage Users
you can delete users

  • Change Logo
To change logo edit logo.html

  • Change style
Edit style.css
