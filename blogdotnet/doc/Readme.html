Requirements

  • Microsoft .NET Framework 2.0
  • MySql 4.0 + as Database Server
  • IIS as Application Server
  • MySql Data Provider

Installation

Premise: don't touch or remove bin directory!! In this directory are some assembly including MySql Data Provider.

By Setup Folder

Recommended: don't rename the folder blogdotnet(Renaming It could bring to unexpeted behavior)
Go to
/blogdotnet/setup/Default.aspx
and follow instructions
See also here Installation

Use of Blog

Detailed information is in Using_blog.html file

The Interface is very intuitive.
To go to admin section go to http://www.yoursite/blogdotnet/admin/Default.aspx, and login with data you have enter during the installation process.

Then you can login automatically going directly to /admin folder http://www.yoursite/blogdotnet/admin/index.aspx
Once you have installed the blog you can
  • Post news and change data in the admin section
  • By login in this section, post comments on the blog, and remove comments
